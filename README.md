# Spell Checker with C++

Dictionary contains 370105 English words.

## How to compile?

### Requirements

- CMake (version >= 3.20)
- Suitable C++ compiler.
- Any build tool CMake supports. (Make or ninja prefered)

### CMake

```sh
cd spell-checker
cmake -S . -B build
cd build
make
```

### Make

```sh
cd spell-checker
make
```

Build binary files locates in `bin` folder in root folder.

```sh
cd ..
cd bin
./spell-checker
```
